# ABSOULUTE CINEMA GENERAL DOCUMENTATION
---
This is a kind of Netflix clone in web, for the final degree project.
---
## FEATURES:
- Browse audiovisual content
  + Searching functionality
  + AI movie suggesting chatbot
  + Interface preferences
- User management functionalities
  + Registration
  + Log-in / log-out
  + Personal Space
  + Favourite contents board
  + Points loyalty system
  + User personalization
- Pricing plans system
- Security system
  + Google auth system
  + Coming soon!
---
## TECHNOLOGIES USED:
- Frontend:
  + JavaScript pure language ( o REACT ya veremos )
  + CSS / Tailwind CSS
  + TMDB API
- Backend:
  + Python language / Django framework
  + SQLite Database
  + REST API
  + WebSocket
 - DevOps
   + Docker
   + CI/CD (Github actions)
